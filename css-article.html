Getting Started With HTML and CSS

Getting Started With HTML and CSS

October 19, 2021

By: Nazar Viznytsya

What is HTML and CSS?

Every website you open has something in common whether it be a complicated website with lots of bells and whistles like Instagram, or a simple website that contains the answer to that one chemistry question on your assignment. The connecting link that makes a website a website is that they all use HTML and CSS. HyperText Markup Language (HTML) is essentially the bones of a website. It broadly defines what elements are in a website. For instance, it can define a heading, a paragraph, an image, or even another website inside yours; However, if you only look at our bones each person seems nearly identical, that’s where Cascading Style Sheets (CSS) come in. CSS styles the website, with CSS you can add colours, borders, transitions, animations, and other aesthetic devices.

Getting Started With HTML

HTML is made up of elements - pieces of a website that each serve a purpose. There are elements for virtually every part of a website, specific elements exist for: images, headings, paragraphs, entire sections, forms, and many others. An element for a paragraph looks like the following:

<p>Some Text</p>

In between the first set of <> we have the name of the element, which is "p", what HTML uses for paragraphs. “<p>” is called the opening tag of the element. The closing tag is “</p>”, and in between the opening and closing tags are the text content of the element. Note, some elements, like an image, don’t have text content within them, these elements don’t have a closing tag, and are just contained in one opening tag.

Some other examples of tags are:

<h2>Some Text</h2>
<div>Some Text</div>
<br>

The “h2” is a tag for the second-largest heading, there exist “h1”-”h6” tags just for headings. The “div” tag is just a container, we’ll discuss it in more detail later. The “br” tag is a break line tag and it acts essentially as an enter key. This is useful as HTML will remove all unnecessary enter keys before running. Notice how it’s a self-closing tag.

A list of elements can be found at: W3Schools

More Advanced HTML

Some elements of websites carry more data than just content text. Images contain links to the image, same with videos, and regular old links contain both content text and a link. We give elements these extra values inside of the opening tag in the form of attributes. For example below is an image tag:

<img src="Image.png">

In this tag the attribute declaring part is src=“Image.png”, here we set the attribute of the source of the image (Called “src” in HTML) to the value “Image.png”. This tells the HTML to display the Image.png file as an image on the website. All attribute declarations follow this format.

Below are some common attribute declarations within elements:

<img src="Image.png" alt="Image Sample">
<button type="submit">
<a href="www.google.com">Link</a>

The first image tag adds an alt element, the “alt” is the text that displays if a user doesn’t have an image and displays only text, usually, this happens if they use a screen reader. The second element is a button, specifically, a submit button which is used extensively in forms. The final element is a link, the “href” is the actual link part which is where you’ll go if you click the link, and the content is what you see as the link on the website.

Nested HTML Tags

HTML elements can also be put into each other for logical ordering. For instance, you might want an element that contains all the links in your navigation menu. To put HTML elements inside of each other put the element to nest inside of the content section:

<div>
    <p>Nested Text</p>
</div>

Here we put a paragraph tag inside of a div container.

Getting Started With CSS

All the HTML we have written right now has nothing more than the basic styling on it, if you want to add custom styling we use CSS. CSS is filled with style declarations, a style declaration looks like the following:

background-color: #000000;

Here we set the background colour style to be the hexadecimal colour black. All CSS declarations follow this basic format.

A full list of style declarations can be found at: W3Schools

Adding CSS to HTML

To add style to an element you have three options, inline, internal, and external.

Inline Styling

For Inline styling you add all your styles into the attribute style on an html element. To set the width and height of our image from before using inline:

<img src="Image.png" style="width:100px; height:100px;">

Here we take our image from last time and we make it have the fixed size of 100px by 100px.

Internal Styling

For Internal styling we use the special HTML tag: "<style></style>" and inside we put our CSS declarations. The declarations in here are a little different because they need to say what element they’re targeting, they will look like:

<style>
    img{
        width: 100px;
        height: 100px;
    }
</style>

The tag before the curly brackets is the target, here we apply the style to all images. If we want to target only some elements we can use classes and IDs. A class is an identifier that you add to multiple elements, you add a class to an element by using the attribute class in an HTML element. You can add multiple classes by separating them with a space. An ID is a label for only one element. We add it with the attribute id to HTML elements.

To reference classes or IDs in an internal stylesheet we use a “.” before the class name or a “#” before an ID name. Implementing all that we’ve learned so far we have:

<style>
    div{
        background-color: black;
    }
    #heading{
        text-transform: uppercase;
    }
    .red-text{
        color: red;
    }
</style>

<div>
    <h1 class="red-text" id="heading">Sample Website</h1>
    <p class="red-text">Lorem Ipsum</p>
</div>

This creates the website:

Here we have an “h1” element and “p” element nested within a “div” element. We style all divs with a black background colour, we assign both texts a class of “red-text” and we make all text for the element with an ID of “heading” to be uppercase.

Notice that to run the HTML file all we do is open the file in a web browser. Note, all HTML files must end with .html, and all CSS files must end with .css

External Styling

Finally to use external CSS, we copy the inner text of the style element into an external file and then we reference it with the following HTML tag:

<link rel="stylesheet" href="index.css">

Here, we import the external stylesheet located in the file index.css

What's Next?

We have now learned the basics of HTML and CSS. We know the structure of HTML and CSS and how to run it. All websites use the following structure which we have defined and we are now well on our way to creating more advanced websites. The best way for you to further your knowledge now is to start making websites. Create ideas of what you want in your website, and then research how to do it if you do not know. This is the way I learned, and hopefully, it will serve you well.
